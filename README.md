# 🐍 Snake Game (Java)

A simple and interactive Snake Game built using **Java Swing**.

## 🎮 Features

* Snake movement using arrow keys
* Food generation system
* Score tracking
* Game over detection
* Smooth gameplay using Java GUI

## 🛠️ Technologies Used

* Java
* Swing (GUI)

## 📁 Project Structure

```
SnakeGame
 └── src
      └── com.lacky.snakegame
           ├── GameFrame.java
           ├── GamePanel.java
           └── Main.java
```

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/varalakshmipurini-bot/snake-game-java.git
```

2. Open project in Eclipse or any Java IDE

3. Run:

```
Main.java
```

## 🎯 Controls

* ⬆️ Up Arrow → Move Up
* ⬇️ Down Arrow → Move Down
* ⬅️ Left Arrow → Move Left
* ➡️ Right Arrow → Move Right
